# `src/ui/shared`

本目录存放可被多个 UI 页面直接复用的前端基础组件与绑定工具。

当前包含：

- `input.ts`：通用防抖文本输入绑定
- `pagination.ts`：通用分页控件渲染与按钮绑定
- `lists.ts`：通用列表空态与键值行渲染
- `metrics.ts`：通用统计值更新
- `tag-pill/`：标签药丸创建与拖拽绑定
- `filter-panel/`：筛选标签与放置区绑定
- `index.ts`：统一导出

设计原则：

- 不依赖具体页面状态结构
- 不直接访问业务仓库或页面专属服务
- 优先提供 DOM 组件与轻量绑定函数，方便不同页面按需组合
