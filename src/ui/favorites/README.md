# 收藏视频功能

## 功能概述

收藏视频功能实现了B站收藏夹的本地同步和管理，支持收藏列表展示、搜索和筛选。

## 主要功能

### 1. 收藏列表
- 展示用户收藏的所有视频
- 显示视频封面、标题、UP主、发布时间和标签
- 支持分页浏览
- 点击视频可跳转到B站观看

### 2. 收藏同步
- 从B站API同步收藏夹数据
- 自动获取视频详细信息
- 确保UP主和标签数据完整
- 支持增量更新

### 3. 收藏搜索
- 按视频标题搜索
- 按标签筛选
- 按UP主筛选
- 支持组合条件查询

### 4. 实时同步
- 监听B站页面的收藏操作
- 自动同步收藏/取消收藏事件
- 实时更新本地收藏数据

## 技术实现

### 后端服务

#### favorite-sync.ts
负责收藏数据的同步和搜索：
- `syncFavoriteVideos`: 同步B站收藏数据
- `searchFavoriteVideos`: 搜索收藏视频
- `ensureCreatorExists`: 确保UP主数据存在
- `ensureTagsExist`: 确保标签数据存在

#### messages.ts
处理收藏相关的消息：
- `favorite_action`: 处理收藏/取消收藏操作
- `sync_favorite_videos`: 触发收藏同步
- `search_favorite_videos`: 执行收藏搜索

### 内容脚本

#### favorite-tracker.ts
监听B站页面的收藏操作：
- 检测收藏按钮状态变化
- 发送收藏事件到后台
- 支持SPA页面URL变化

### 前端页面

#### favorites.html
收藏列表页面的HTML结构

#### favorites.css
收藏列表页面的样式

#### favorites.js
收藏列表页面的交互逻辑：
- 加载和展示收藏视频
- 处理同步和搜索操作
- 管理分页和筛选

## 使用方式

### 首次使用
1. 确保已登录B站并获取用户ID
2. 点击"同步收藏"按钮
3. 等待同步完成
4. 查看收藏视频列表

### 日常使用
1. 在B站页面收藏视频时自动同步
2. 使用搜索框查找特定视频
3. 使用标签和UP主筛选器过滤视频
4. 点击视频跳转观看

## 数据存储

所有收藏数据存储在IndexedDB中：
- `videos`: 视频基本信息
- `collections`: 收藏夹信息
- `collection_items`: 收藏项记录
- `creators`: UP主信息
- `tags`: 标签信息

## 注意事项

1. 首次同步可能需要较长时间，取决于收藏视频数量
2. 同步过程中请勿关闭浏览器
3. 搜索功能支持模糊匹配
4. 分页大小默认为20条/页
